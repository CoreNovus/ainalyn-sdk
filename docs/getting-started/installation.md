# Installation

## Requirements

- Python 3.11, 3.12, or 3.13
- pip

## Install from PyPI

```bash
pip install ainalyn-sdk
```

## Install from Source

For development:

```bash
git clone https://github.com/CoreNovus/ainalyn-sdk.git
cd ainalyn-sdk
pip install -e ".[dev]"
```

## Verify Installation

Check the SDK is installed:

```bash
ainalyn --version
```

Test Python import:

```python
python -c "import ainalyn; print('OK')"
```
